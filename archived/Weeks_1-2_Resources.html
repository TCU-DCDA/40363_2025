Week 1: Course Introduction & Setup (Labs 1-2)

Interactive Tutorial

MDN Web Docs: HTML Basics Beginner

Mozilla's comprehensive guide to HTML fundamentals. This is the gold standard for web development documentation and is completely free.

Key Topics:
  • HTML document structure and syntax
  • Essential HTML elements (headings, paragraphs, lists)
  • Semantic HTML and why it matters
  • File organization for web projects
Lab Connections: Perfect preparation for Lab 1 (HTML structure) and Lab 2 (file organization). Reference this during your first HTML file creation.
Video Tutorial

VS Code Official Getting Started Videos Beginner

Microsoft's official video series covering VS Code basics, extensions, and productivity features. Short, focused videos perfect for beginners.

Essential Videos:
  • "Getting Started" (5 min) - Interface overview
  • "Code Editing" (6 min) - Essential shortcuts and features
  • "Extensions" (4 min) - Installing and managing extensions
  • "Debugging" (8 min) - Basic debugging concepts
Lab Connections: Essential for Lab 1 VS Code setup and workflow optimization. Watch before your first lab session.
Reference Guide

web.dev: Learn HTML Beginner

Google's comprehensive HTML learning path with modern best practices, accessibility focus, and hands-on examples.

Standout Features:
  • Modern HTML5 semantic elements explained
  • Accessibility considerations from the start
  • Interactive code examples you can edit
  • Progressive difficulty with clear explanations
Lab Connections: Excellent supplement for Labs 1-2. The accessibility focus aligns perfectly with our semantic HTML emphasis.
Interactive Practice

freeCodeCamp: Responsive Web Design (HTML Sections) Beginner

Hands-on coding challenges that reinforce HTML concepts through practice. Complete the first few sections for HTML fundamentals.

Recommended Sections:
  • "Learn HTML by Building a Cat Photo App" - Perfect first project
  • "Learn Basic HTML and HTML5" - Comprehensive element coverage
  • Interactive coding environment with instant feedback
  • Progress tracking and certificates (optional)
Lab Connections: Great practice between Lab 1 and Lab 2. The cat photo app project mirrors your personal webpage development.

Week 2: HTML Foundations & Git Introduction (Labs 3-4)

Interactive Tutorial

Learn Git Branching Beginner

Visual, interactive Git tutorial that makes version control concepts clear through hands-on practice. Perfect introduction to Git workflows.

Key Features:
  • Visual representation of Git operations
  • Step-by-step tutorials with immediate feedback
  • Covers basic commands: add, commit, push, pull
  • No installation required - runs in browser
Lab Connections: Essential preparation for Lab 3 Git introduction. Complete the first 3-4 levels before class.
Video Series

Git and GitHub for Beginners - Crash Course (freeCodeCamp) Beginner

Comprehensive 1-hour video covering Git basics, GitHub setup, and workflows. Perfect for visual learners who want to see the process.

Topics Covered:
  • What is Git and why use version control?
  • GitHub account setup and repository creation
  • Basic Git commands and workflow
  • Connecting local projects to GitHub
Lab Connections: Watch before Lab 3 to understand the "why" behind Git. Covers exactly the workflow you'll practice.
Reference Guide

HTML Reference Beginner

Beautiful, visual reference for all HTML elements with examples and browser support. Perfect quick-reference tool for Labs 3-4.

Why It's Great:
  • Visual examples of every HTML element
  • Semantic HTML5 elements clearly explained
  • Clean, searchable interface
  • Shows element relationships and nesting
Lab Connections: Bookmark this for Labs 3-4 when you're enhancing HTML structure and implementing semantic elements.
Documentation

GitHub Hello World Guide Beginner

GitHub's official tutorial for creating your first repository, making commits, and understanding the GitHub workflow.

Learn About:
  • Creating repositories and README files
  • Making and committing changes
  • GitHub web interface vs. desktop tools
  • Collaboration basics (for future reference)
Lab Connections: Perfect companion to Lab 3. Follow along to reinforce the repository setup and commit process.
