Git Workflow Setup

Work in your existing personal-website-foundation repository from previous labs.

Tasks

1. Resources Section (External Links)
  1. Add a new section named Resources with an unordered list of 3 external links.
  2. Each link uses descriptive text (no raw URLs) and unique domains.
  3. Add target="_blank" and rel="noopener" for external links as needed.
2. Internal Jump Link
  1. Add id to a lower-page heading (e.g., id="skills").
  2. From your top nav or intro, add an internal link to that ID (e.g., <a href="#skills">Skip to skills</a>).
  3. Test the jump in the browser.
3. Image Audit
  1. Review current images and identify purely decorative ones.
  2. For decorative images, plan to replace with CSS backgrounds later; add an HTML comment <!-- candidate for CSS background -->.
  3. Ensure meaningful images include concise, descriptive alt text.
4. Add a Second Figure
  1. Add a new <figure> containing <img> and <figcaption>.
  2. Keep caption ‚â§ 12 words and relevant to the image.
  3. Verify image path and loading; test with a placeholder if needed.

Link Rules

  • External links use rel="noopener" when target="_blank" is present.
  • No raw URLs as link text; write human-friendly, descriptive labels.
