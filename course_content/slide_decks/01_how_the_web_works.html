What is a Browser?

A browser is your window to the internet

  • Handles requests: Asks servers for websites and resources

What is a Server?

A server is a computer that serves websites to other computers

What Servers Do

  • Store website files
  • Process requests
  • Send responses
  • Manage databases
  • Handle user accounts

Types of Servers

  • Web servers (Apache, Nginx)
  • Database servers (MySQL)
  • File servers
  • Email servers
  • Cloud servers (AWS, Azure)

HTTP vs HTTPS

HTTP
vs
HTTPS

HTTP

HyperText Transfer Protocol

  • Basic web communication
  • Data sent in plain text
  • Not secure
  • Can be intercepted
  • No encryption

HTTPS

HTTP Secure

  • Encrypted communication
  • Data is protected
  • Secure connection
  • Cannot be easily intercepted
  • Required for sensitive data

HTTP: Sending a postcard - anyone can read it

HTTPS: Sending a letter in a sealed envelope

Putting It All Together

The Complete Web Address

https://www.example.com/page.html

  • https:// - The protocol (secure communication)
  • www - The subdomain (optional)
  • example - The domain name (unique identifier)
  • .com - The top-level domain (category)
  • /page.html - The specific file or page
Think of it like following a path to a specific location:
USA → Texas → Fort Worth → 76129 → TCU → Reed Hall → Room 219
https:// → www. → tcu → .edu → /academics → /page.html

Both are paths! One guides you through physical space, the other guides your browser through the internet to find the exact page you want.

Why This Matters for Web Development

  • Understanding your audience: Know how people access your sites
  • Testing across browsers: Your site should work everywhere
  • Choosing hosting: Where will your server be located?
  • Domain strategy: Picking the right web address
  • Security planning: Always use HTTPS for user trust
  • Performance optimization: How servers affect loading speed

Remember: As a web developer, you're creating experiences that travel through this entire system to reach your users. Understanding each part helps you build better websites.
