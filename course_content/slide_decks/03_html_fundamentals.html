Creating Lists

Unordered Lists (Bullets)

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Water</li>
</ul>

Result:

  • Coffee
  • Tea
  • Water

Ordered Lists (Numbers)

<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Eat breakfast</li>
</ol>

Result:

  1. Wake up
  2. Brush teeth
  3. Eat breakfast

Creating Links

Links are the connections between web pages

<!-- Link to another website -->
<a href="https://www.google.com">Visit Google</a>
 
<!-- Link to another page on your site -->
<a href="about.html">About Me</a>
 
<!-- Email link -->
<a href="mailto:your@email.com">Send Email</a>

Key Points:

  • href = "hypertext reference" (where the link goes)
  • Text between tags = what visitors see and click
  • Always use descriptive text (not "click here")

Adding Images

<img src="photo.jpg" alt="A beautiful sunset over the mountains">
 
<!-- Images from the web -->
<img src="https://example.com/image.jpg" alt="Description">

üîó Important Image Attributes:

src = "source" - where the image file is located
alt = "alternative text" - describes the image for screen readers
width = controls image width (optional)
height = controls image height (optional)

Why alt text matters: Screen readers use it to describe images to visually impaired users. Always include meaningful descriptions!

Attributes: Extra Information for Elements

Attributes provide additional information about HTML elements

<p id="intro" class="highlight">Important paragraph</p>
 
<a href="contact.html" title="Go to contact page">Contact</a>
 
<img src="logo.png" alt="Company logo" width="200">
