CSS = Cascading Style Sheets

The language for describing how HTML elements should look

CSS Syntax: The Basic Pattern

/* CSS Rule Structure */
selector {
  property: value;
  property: value;
}
 
/* Real Example */
h1 {
  color: blue;
  font-size: 2rem;
  text-align: center;
}

üéØ Key Parts:

Selector: Which HTML elements to style
Property: What aspect to change (color, size, etc.)
Value: How to change it (blue, 2rem, center, etc.)
Semicolon: Ends each property-value pair

Three Ways to Add CSS

üîó Method 1: External CSS File (BEST)

<!-- In your HTML head -->
<link rel="stylesheet" href="style.css">

Benefits: Organized, reusable, faster loading

üìù Method 2: Internal CSS (OKAY)

CSS Selectors: Targeting Elements

Element Selector: h1, p, img
Class Selector: .highlight, .button
ID Selector: #header, #main-title
/* Style all paragraphs */
p {
  color: #333;
}
 
/* Style elements with class="highlight" */
.highlight {
  background-color: yellow;
}
 
/* Style element with id="main-title" */
#main-title {
  font-size: 3rem;
}

Working with Colors

üé® Color Format Options:

Named Colors: red, blue, green, white, black
color: red;
Hex Colors: #RRGGBB format
color: #3498db;
RGB Colors: Red, Green, Blue values
color: rgb(46, 204, 113);
h1 {
  color: #2c3e50;
  /* Text color */
  background-color: #ecf0f1;
  /* Background color */
}

Styling Text

h1 {
  font-family: 'Arial', sans-serif;
  font-size: 2.5rem;
  font-weight: bold;
  text-align: center;
  line-height: 1.4;
}
 
p {
  font-size: 1.1rem;
  line-height: 1.6;
  text-decoration: none;
}

üìù Text Property Quick Reference:

  • font-family: Choose the typeface
  • font-size: Size of text (px, rem, em)
  • font-weight: bold, normal, 100-900
  • text-align: left, center, right, justify
  • line-height: Space between lines

The CSS Box Model

Every HTML element is a rectangular box

Margin
Border
Padding
Content Area
.card {
  margin: 20px;
  /* Space outside the element */
  border: 2px solid #ddd;
  /* Border around the element */
  padding: 15px;
  /* Space inside the element */
  width: 300px;
  /* Content area width */
}

Controlling Spacing

/* Individual sides */
.header {
  margin-top: 20px;
  margin-bottom: 30px;
  padding-left: 15px;
  padding-right: 15px;
}
 
/* Shorthand properties */
.content {
  margin: 20px 10px;
  /* top/bottom: 20px, left/right: 10px */
  padding: 15px 20px 10px 5px;
  /* top, right, bottom, left */
}

üéØ Spacing Tips:

  • margin: Creates space between elements
  • padding: Creates space inside elements
  • Use rem/em for responsive spacing
  • margin: 0 auto; centers block elements
