What JavaScript Actually Does

JavaScript makes websites respond to users:

  • Click a button: Something happens (forms submit, menus open, content changes)
  • Type in a search box: Suggestions appear instantly
  • Scroll down: New content loads automatically
  • Hover over an image: It zooms or changes
  • Fill out a form: Errors are caught before you submit
  • Play a video: Controls respond to your clicks

Think of a smartphone:

• Hardware = HTML (the structure)

• Design = CSS (how it looks)

• Apps & Software = JavaScript (what it can do)

Without JavaScript, websites are like phones that can only display pictures!

Functions: Code Recipes

Functions are sets of instructions you can use over and over

Input
("Alice")
→
greetStudent()
Add "Hello" + name + "!"
→
Output
("Hello Alice!")
function greetStudent(name) {
  return 'Hello ' + name + '!';
}
// Use the function:
greetStudent('Alice'); // Returns "Hello Alice!"

Like a recipe for cookies: Write the recipe once, then follow it whenever you want cookies. Functions are recipes for code tasks.

Let's Try It: Browser Console

The browser console is your JavaScript playground

How to Open Console

  • Windows: F12 or Ctrl+Shift+I
  • Mac: Cmd+Option+I
  • Any browser: Right-click → Inspect → Console

Try These Commands

> console.log('Hello World!')
Hello World!
> 2 + 3
5
> let name = 'Your Name'
> console.log('Hi ' + name)
Hi Your Name

Like a calculator for code: Type JavaScript, see results immediately. Perfect for learning and testing ideas!

Common Beginner Mistakes

Everyone makes these mistakes - here's how to fix them:

❌ Forgetting Quotes
let name = Alice;
Error: Alice is not defined
✅ Use Quotes for Text
let name = 'Alice';
✓ Works perfectly!
❌ Wrong Capitalization
let studentname = 'Alice';
console.log(studentName);
Error: studentName is not defined
✅ Match Exactly
let studentName = 'Alice';
console.log(studentName);
Alice

JavaScript is case-sensitive and picky about syntax - but error messages help you learn!
