Section vs Article vs Div

üß© When to Use Each:

<section> - Themed groups of content

Example: "About Me", "My Skills", "Contact"

<article> - Standalone, complete content

Example: Blog posts, portfolio projects

<div> - Generic container for styling

Example: Layout boxes, style wrappers

<section id="projects">
  <h2>My Projects</h2>
  <article>
    <h3>Portfolio Website</h3>
    <p>My first web development project...</p>
  </article>
</section>

Proper Heading Hierarchy

Headings create a table of contents for your page

Sarah's Portfolio (h1)

About Me (h2)

My Projects (h2)

Portfolio Website (h3)

JavaScript Game (h3)

Contact (h2)

üìù Heading Rules:

  • One h1 per page - your main title
  • Don't skip levels - h1 ‚Üí h2 ‚Üí h3 (not h1 ‚Üí h3)
  • Use for structure, not styling - CSS handles appearance
